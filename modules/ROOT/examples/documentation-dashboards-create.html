Create a new documentation dashboard

The creation of a custom Documentation Dashboard helps to visualize the effectiveness of published documentation based on topic type (concept, task, reference).

Prerequisites

Prerequisites

Navigation path

Home > Dashboards > Create New

Task

  1. Click Add Report.
  2. From the Report Type dropdown, select a topic type and enable the chekbox next to each documentation file you want to evaluate:
  3. Specify a report code, name, and description

    4. Note:Report code must be unique

  4. Build the report by dragging and dropping data objects, noting the following about Categories, Sorts, Filters, and Layout tabs:
    1. Categories: Allows the user to select the data categories (RTFM-Industries tables) that should be accessible on the report. One or more categories must be selected and will determine the fields that are available to select on the other tabs.
      Note: Users will see the tables they're permissioned to access.
    2. Sorts: Allows the user to select one or more data fields used to sort the data. This is optional
    3. Filters: Allows the user to select one or more data fields used to filter the data. This is optional
    4. Layout: Allows the user to select and order the data fields that should display on the report
  5. To subscribe others to this Documentation Dashboard, enter their email address in the Subscriber field.
  6. Click Save when finished.

  7. Result: the Documentation Dashboard is added to the list in the bottom panel.
