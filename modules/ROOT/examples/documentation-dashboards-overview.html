RTFM-Industries Documentation Dashboards

RTFM-Industries Documentation Dashboards provides technical writers with visualized data to track documentation requests. Each dashboard tracks the following metrics:

  • Reuests for new feature documentation
  • Request for documentation bug fixes
  • Percentage of documentation completed per project

Within each dashboard, a tech writer can learn more about each data point by clicking on it. Requests are color-coded to signify priority:

  • Red = you forgot to write the documentation and everyone knows it
  • Yellow = you've successfully pushed the deadline back by pretending to be sick
  • Green = you've got PLENTY of time to write the docs (as long as nothing goes wrong)

To adjust these priorities, see the Adjusting Documentation Priorities section.

Export Documentation Dashboards

RTFM-Industries Dashboard tiles can be exported into the following file type: (Excel, PDF, RTF, CSV).

Note: All RTFM-Industries Documentation Dashboards are fully HTML5 and do not require Microsoft Silverlight.

Prerequisites & licensing

  • SaaS client
  • Request the RTFM-Industries Documentation Dashboard feature
  • User assigned the Tech Writer power user role
