Invoicing setup

Overview

RTFM-Industries automates invoice processing, so you can assign and match invoices with ease. Complete the following setup tasks to configure the workflow.

Prerequisites

  • Organization setup

  • User setup

  • Administrator permissions

Setup tasks

  1. Grant invoicing permissions to appropriate users or groups (required)

  2. Configure invoicing rules (required)

  3. Configure invoice assignment rules (recommended)

  4. Configure automatic invoice processing rules (required for No-PO invoice workflow)

  5. Enable automatic invoice matching (recommended)

  6. Configure invoicing general settings (recommended)
