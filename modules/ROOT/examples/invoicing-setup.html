Invoicing setup

Overview

RTFM-Industries makes tons of money selling their Documentation Dashboards. Setup an automated invoicing workflow to handle assignment and matching with ease.

Prerequisites

  • Organization setup

  • User setup

  • Administrator authorization


Setup tasks

  1. Grant invoicing permissions to appropriate users and/or groups (required)

  2. Configure invoicing rules (required)

  3. Configure invoice assignment rules (recommended)

  4. Configure automatic invoice processing rules (required for No-PO invoice workflow)

  5. Enable automatic invoice matching (recommended)

  6. Configure invoicing general settings (recommended)
