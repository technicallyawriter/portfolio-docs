Invoicing setup

Overview

Complete the following administrator setup tasks to configure rules and settings to automate common invoice processing workflows that align with your organization's rules and preferences. Additionally, ensure the appropriate users are granted your preferred invoicing authorization.

Prerequisites

  • Organization setup

  • User setup

  • Administrator authorization


Setup tasks

  1. Grant invoicing authorizations to appropriate users and/or groups (required)

  2. Configure invoicing rules (required)

  3. Configure supplier invoice assignment rules (recommended)

  4. Configure automatic invoice processing rules (required for No-PO invoice workflow)

  5. Enable automatic invoice matching (recommended)

  6. Configure invoicing general settings (recommended)
